This repository contains the official website of the ACSL Race League, a competitive sim racing championship built around Assetto Corsa. The website provides standings, the race calendar, articles, results, and league updates. All content is optimized for clarity, performance, and search engine visibility.

## Purpose
The ACSL Race League website serves as the central platform for drivers and followers of the league. It consolidates all season information, announcements, and documentation in one place. The project is tailored specifically for the ACSL community and is not intended for reuse or redistribution.

## Features
- Standings with automatic updates  
- Full race calendar with event details  
- Articles and league announcements  
- Responsive layout for desktop and mobile  
- Built using lightweight HTML, CSS, and JavaScript
